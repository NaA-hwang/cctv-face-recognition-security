# 📸 Normal02 폴더 - 이지선의 얼굴 사진을 저장하세요

## 필수 파일들 (5장):
1. `normal02_front_1.jpg` - 정면 사진 1 (무표정)
2. `normal02_front_2.jpg` - 정면 사진 2 (미소 또는 다른 표정)  
3. `normal02_left_45_1.jpg` - 왼쪽 45도 측면
4. `normal02_right_45_1.jpg` - 오른쪽 45도 측면
5. `normal02_up_angle_1.jpg` - 위에서 내려다본 각도 (CCTV 시각)

## 특별 참고사항 (이지선 - 39세 여성, 운동선수):
- **흑발 긴머리**: 특징적인 긴 머리가 잘 보이도록 촬영
- **운동선수**: 스포츠 웨어나 운동복 착용 가능
- **중년 여성**: 39세 여성의 특징
- **일반인**: 밥도둑이라는 농담성 전과, 실제 위험도 낮음

## 촬영 가이드:
- **해상도**: 최소 640x480, 권장 1280x720
- **조명**: 자연광 또는 밝은 실내조명  
- **배경**: 단색 배경 (흰색, 회색 권장)
- **긴 머리 강조**: 특징적인 긴 머리가 선명하게 보이도록

모든 사진을 넣으신 후 `python scripts/process_face_data.py`를 실행하여 데이터를 처리하세요!
