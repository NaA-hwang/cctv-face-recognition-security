# 📸 Normal03 폴더 - 순대국의 얼굴 사진을 저장하세요

## 필수 파일들 (5장):
1. `normal03_front_1.jpg` - 정면 사진 1 (무표정)
2. `normal03_front_2.jpg` - 정면 사진 2 (미소 또는 다른 표정)  
3. `normal03_left_45_1.jpg` - 왼쪽 45도 측면
4. `normal03_right_45_1.jpg` - 오른쪽 45도 측면
5. `normal03_up_angle_1.jpg` - 위에서 내려다본 각도 (CCTV 시각)

## 특별 참고사항 (순대국 - 54세 여성, 쉐프):
- **다듬지 않은 눈썹**: 특징적인 눈썹 모양 강조
- **쉐프**: 요리사 관련 의상이나 모자 착용 가능
- **중년 여성**: 54세 여성의 특징
- **일반인**: 낮은 위험도, 대조군 역할

## 촬영 가이드:
- **해상도**: 최소 640x480, 권장 1280x720
- **조명**: 자연광 또는 밝은 실내조명  
- **배경**: 단색 배경 (흰색, 회색 권장)
- **눈썹 강조**: 특징적인 눈썹 모양이 선명하게 보이도록

모든 사진을 넣으신 후 `python scripts/process_face_data.py`를 실행하여 데이터를 처리하세요!
