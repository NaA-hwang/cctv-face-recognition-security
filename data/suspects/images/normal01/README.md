# 📸 Normal01 폴더 - 하니짱의 얼굴 사진을 저장하세요

## 필수 파일들 (5장):
1. `normal01_front_1.jpg` - 정면 사진 1 (무표정)
2. `normal01_front_2.jpg` - 정면 사진 2 (미소 또는 다른 표정)  
3. `normal01_left_45_1.jpg` - 왼쪽 45도 측면
4. `normal01_right_45_1.jpg` - 오른쪽 45도 측면
5. `normal01_up_angle_1.jpg` - 위에서 내려다본 각도 (CCTV 시각)

## 특별 참고사항 (하니짱 - 28세 남성, 간호사):
- **짧은 머리**: 깔끔한 짧은 머리 스타일
- **간호사**: 의료진 유니폼 착용 가능
- **젊은 남성**: 28세 남성의 특징
- **일반인**: 무단횡단 정도의 경미한 전과만 있음

## 촬영 가이드:
- **해상도**: 최소 640x480, 권장 1280x720
- **조명**: 자연광 또는 밝은 실내조명  
- **배경**: 단색 배경 (흰색, 회색 권장)
- **머리 스타일**: 깔끔한 짧은 머리가 잘 보이도록

모든 사진을 넣으신 후 `python scripts/process_face_data.py`를 실행하여 데이터를 처리하세요!
