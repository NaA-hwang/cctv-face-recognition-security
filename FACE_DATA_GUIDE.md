# 📸 팀원 얼굴 데이터 수집 가이드

## 🎯 개요
CCTV 용의자 식별 시스템을 위해 **4명 팀원**의 얼굴 사진을 수집하고 AI 학습 데이터로 처리하는 방법을 안내합니다.

## 👥 팀원 구성 및 저장 폴더
```
data/suspects/images/
├── hwang_yunha/     # 황윤하 (37세, 여성, 앞머리, 절도범)
├── sundaeguk/       # 순대국 (54세, 여성, 쉐프, 다듬지않은눈썹)
├── hanijjang/       # 하니짱 (28세, 남성, 간호사, 짧은머리)  
└── leejisun/        # 이지선 (39세, 여성, 운동선수, 긴머리)
```## 📋 각 팀원별 필수 사진 (5장)

### 🔹 필수 파일 이름 (정확히 입력):
1. **`front_1.jpg`** - 정면 사진 1 (무표정)
2. **`front_2.jpg`** - 정면 사진 2 (미소 또는 다른 표정)  
3. **`left_45_1.jpg`** - 왼쪽 45도 측면
4. **`right_45_1.jpg`** - 오른쪽 45도 측면
5. **`up_angle_1.jpg`** - 위에서 내려다본 각도 (CCTV 시각)

## 📸 촬영 조건

### 🔧 기술적 요구사항:
- **해상도**: 최소 640x480, 권장 1280x720
- **파일 형식**: JPG 또는 PNG  
- **파일 크기**: 500KB ~ 5MB 권장
- **색상**: 컬러 이미지

### 💡 촬영 환경:
- **조명**: 자연광 또는 밝은 실내조명
- **배경**: 단색 배경 (흰색, 회색 권장)
- **거리**: 얼굴이 화면의 30-50% 차지하도록
- **화질**: 얼굴 특징이 선명하게 보이도록

### 📐 촬영 각도:
```
정면 (front_1, front_2):     왼쪽 45도:        오른쪽 45도:       위에서 30도:
     👤                        👤              👤               👤  
     정면                      ◄45°            45°►             ▲30°
                                                                (CCTV 시점)
```

## 🎭 각 팀원별 촬영 포인트

### 🧑‍🎓 홍길동 (20대, 학생)
- **안경**: 반드시 안경 착용한 상태로 촬영
- **옷차림**: 캐주얼한 학생 스타일
- **머리**: 짧은 머리, 깔끔한 스타일
- **특징**: 젊은 얼굴, 학생다운 분위기

### 👔 김철수 (30대, 회사원)
- **정장**: 셔츠나 정장 차림
- **머리**: 단정한 남성 헤어스타일
- **특징**: 성숙한 얼굴, 직장인 분위기
- **표정**: 진중하고 신뢰감 있는 표정

### 👩‍🏫 박영희 (40대, 교사)  
- **긴 머리**: 특징적인 긴 머리가 잘 보이도록
- **옷차림**: 단정한 교사 스타일
- **특징**: 지적이고 온화한 인상
- **포즈**: 측면 사진에서 머리 스타일이 잘 드러나도록

### 🧔 이민수 (일반인)
- **수염**: 특징적인 수염이 잘 보이도록 촬영
- **캐주얼**: 일상복 또는 캐주얼 의상
- **특징**: 수염과 중간 체격이 드러나도록
- **각도**: 수염이 잘 보이는 각도로 촬영

## 🔄 데이터 처리 과정

### 1단계: 사진 저장
```bash
# 각 팀원의 폴더에 5장씩 저장
data/suspects/images/hong_gildong/
├── front_1.jpg
├── front_2.jpg  
├── left_45_1.jpg
├── right_45_1.jpg
└── up_angle_1.jpg
```

### 2단계: AI 처리 실행
```bash
# 얼굴 임베딩 생성 및 데이터베이스 구축
python scripts/process_face_data.py
```

### 3단계: 처리 결과 확인
```bash
# 생성된 파일들:
data/embeddings/face_embeddings.db    # 임베딩 데이터베이스
data/processed/face_vectors.json     # 백업용 JSON
logs/processing_log.txt              # 처리 로그
```
