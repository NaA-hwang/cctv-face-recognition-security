## 🚀 **프로젝트 개요**
RetinaFace + ArcFace AI 모델을 활용한 실시간 CCTV 용의자 식별 시스템

## ⚠️ **현재 구현 상태: 프레임워크만 구축됨 (실제 AI 기능 미구현)**

---

## ===============================================================================
## **🔥 핵심 구현 필요 사항 (우선순위 순)**
## ===============================================================================

### **1. AI 라이브러리 설치 [필수]**
```bash
pip install insightface onnxruntime opencv-python
```
❌ **현재 상태: InsightFace 미설치**
✅ **목표: 얼굴 인식 모델 로드 가능**

### **2. 실제 팀원 얼굴 데이터 수집 [필수]**
```
data/suspects/images/
├── hwang_yunha/     # 황윤하 사진 5-10장
├── sundaeguk/       # 순대국 사진 5-10장  
├── hanijjang/       # 하니짱 사진 5-10장
└── leejisun/        # 이지선 사진 5-10장
```
❌ **현재 상태: README 파일만 있음 (실제 사진 없음)**
✅ **목표: 각 팀원당 다양한 각도 사진 5-10장**

### **3. 얼굴 임베딩 데이터베이스 구축 [필수]**
```bash
python scripts/process_face_data.py
```
❌ **현재 상태: 스크립트 있지만 InsightFace 없어서 실행 불가**
✅ **목표: 실제 얼굴 특징 벡터 추출 및 DB 저장**

### **4. 백엔드 API 실제 구현 [필수]**
- `backend/api/detect.py` - 실시간 얼굴 감지 로직
- `backend/api/upload.py` - 비디오 처리 로직  
- `backend/app.py` - AI 모델 초기화 및 연동

❌ **현재 상태: API 엔드포인트만 정의됨 (실제 처리 로직 없음)**
✅ **목표: 실제 AI 모델과 연동된 처리 파이프라인**

## ===============================================================================
## **⚡ 빠른 시작 가이드 (구현 후)**
## ===============================================================================

1. **환경 설정**
```bash
pip install -r requirements.txt
```

2. **데이터 준비**
```bash
# 각 팀원 사진을 data/suspects/images/ 폴더에 저장
# 가이드 참조: data/FACE_DATA_GUIDE.md
```

3. **AI 모델 초기화**
```bash
python scripts/process_face_data.py
```

4. **서버 실행**
```bash
python backend/app.py
```
